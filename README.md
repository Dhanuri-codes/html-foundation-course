# 📘 HTML Foundation Course

This is a beginner-friendly HTML learning project built as part of my web development practice.  
It covers the basic structure and fundamentals of HTML in a simple and easy-to-understand format.

---

## 🌐 Live Demo
👉 https://dhanuri-codes.github.io/html-foundation-course/

## 📌 Project Purpose
The goal of this project is to:
- Practice HTML structure and elements
- Learn how to build a simple educational website
- Understand headings, paragraphs, lists, and layout basics
- Improve frontend development skills
